# Debug vs Release 版本差异对比

## 界面差异

| 界面元素 | Debug版本 | Release版本 |
|---------|-----------|-------------|
| **顶部帮助提示** | ✅ 显示操作提示 | ❌ 隐藏 |
| **底部控制栏** | ✅ 完整显示 | ✅ 显示（部分按钮隐藏） |
| **设置按钮 ⚙** | ✅ 显示 | ❌ 隐藏 |
| **播放/暂停按钮** | ✅ 显示 | ✅ 显示 |
| **扫描U盘按钮** | ✅ 显示 | ✅ 显示 |
| **U盘状态指示** | ✅ 显示 | ✅ 显示 |

## 播放器菜单差异（长按播放器弹出）

| 菜单选项 | Debug版本 | Release版本 |
|---------|-----------|-------------|
| **播放/暂停** | ✅ 显示 | ✅ 显示 |
| **静音/取消静音** | ✅ 显示 | ✅ 显示 |
| **选择文件 📁** | ✅ 显示 | ❌ 隐藏 |
| **扫描本地媒体 🔍** | ✅ 显示 | ❌ 隐藏 |
| **清除内容 🗑** | ✅ 显示 | ❌ 隐藏 |
| **音量信息** | ✅ 显示 | ✅ 显示 |

## 功能差异

| 功能 | Debug版本 | Release版本 |
|------|-----------|-------------|
| 进入设置页面 | ✅ 可以 | ❌ 不能 |
| 手动选择文件播放 | ✅ 可以 | ❌ 不能 |
| 扫描本地媒体 | ✅ 可以 | ❌ 不能 |
| 清除播放器内容 | ✅ 可以 | ❌ 不能 |
| 调试日志输出 | ✅ 开启 | ❌ 关闭 |
| U盘自动拷贝播放 | ✅ 支持 | ✅ 支持 |
| 播放控制 | ✅ 支持 | ✅ 支持 |
| 音量控制 | ✅ 支持 | ✅ 支持 |

## 总结

- **Debug版本** = 开发调试用，全功能开放
- **Release版本** = 给客户用，只保留核心播放功能，界面简洁

## 如何为客户开启更多功能

在U盘根目录放置 `btf_config.json` 配置文件，示例：

```json
{
  "version": 1,
  "features": {
    "showSettingsButton": true,
    "allowManualFileSelect": true,
    "allowLocalMediaScan": true,
    "showHelpTip": true
  }
}
```
