# Project Intent

## Product Goal

本项目是一个面向 **RK3568 / Android 11+** 设备的 HDMI 多窗口媒体播放器，目标是以 **客户交付设备 / kiosk / 信发终端** 形态稳定运行。

核心业务能力：

1. 开机启动应用
2. 启动后恢复本地缓存媒体
3. 检测 U 盘、读取 `btf_config.json`
4. 将 U 盘媒体复制到本地缓存
5. 按布局进行 1~4 路媒体播放
6. 接收外部广播命令控制播放器
7. 向受控接收方回传状态

## Non-Goals

- 不是通用消费级播放器
- 不是自由安装第三方插件的平台
- 不是以“开发模式全开放”为最终客户交付形态

## Delivery Standard

只有同时满足以下条件才算“可交付”：

- Release 行为与文档一致
- 远控链路有权限边界
- U 盘链路可恢复、可验证
- build / test / lint 有可重复证据
- 文档能支撑部署、验收、运维
