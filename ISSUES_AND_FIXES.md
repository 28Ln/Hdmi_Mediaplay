# 代码问题清单与修复计划

## 严重问题 (导致崩溃) - 已修复 ✅

### 1. ✅ MainActivity.setupFullscreen() 空指针异常
- **问题**: `window.insetsController` 在 `setContent` 之前调用可能为 null
- **修复**: 使用 WindowCompat，移到 LaunchedEffect 中

### 2. ✅ ExoPlayer 内存泄漏
- **问题**: VideoPlayerView 中 ExoPlayer 可能在 Composable 重组时创建多个实例
- **修复**: 添加生命周期管理，正确释放资源

### 3. ✅ Service 未正确处理 null Intent
- **问题**: UsbMonitorService.onStartCommand 未实现
- **修复**: 添加 onStartCommand 实现

### 4. ✅ 广播接收器注册异常
- **问题**: Android 13+ 需要指定 RECEIVER_EXPORTED 标志
- **修复**: 添加 RECEIVER_NOT_EXPORTED 标志

### 5. ✅ 文件路径空指针
- **问题**: VideoPlayerView 传入空路径时会崩溃
- **修复**: 添加空路径和文件存在性检查
