# 图片裁剪工具 - 任务清单

## 功能说明
用于屏幕拼接场景，将一张大图自动裁剪成多块，分配到各个播放器显示，实现多屏拼接效果。

## 支持的布局

| 布局 | 说明 | 适用图片比例 |
|------|------|-------------|
| 1x2 | 1行2列 | 2:1 横向 |
| 2x1 | 2行1列 | 1:2 纵向 |
| 1x3 | 1行3列 | 3:1 横向 |
| 3x1 | 3行1列 | 1:3 纵向 |
| 2x2 | 2行2列 | 1:1 方形 |
| 1x4 | 1行4列 | 4:1 横向 |
| 4x1 | 4行1列 | 1:4 纵向 |

## 裁剪逻辑

```
示例: 5760x1080 图片，选择 1x3 布局
- 每块宽度 = 5760 / 3 = 1920
- 每块高度 = 1080
- 块1: (0, 0, 1920, 1080) -> player1
- 块2: (1920, 0, 3840, 1080) -> player2  
- 块3: (3840, 0, 5760, 1080) -> player3
```
